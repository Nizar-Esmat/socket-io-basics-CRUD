# ⚡ LivePosts – Real-time CRUD with Socket.IO

[![Postman Docs](https://img.shields.io/badge/Postman-View%20Docs-orange?logo=postman)](https://documenter.getpostman.com/view/45359829/2sBXqFNhqF)

A full-stack app demonstrating live create, read, update, delete, and search operations using **Socket.IO**, **Express**, **MongoDB**, and a vanilla **HTML/Tailwind/jQuery** frontend.

📬 **Postman Docs:** https://documenter.getpostman.com/view/45359829/2sBXqFNhqF

---

## 📦 Stack

| Layer    | Tech                          |
|----------|-------------------------------|
| Backend  | Node.js, Express, Socket.IO   |
| Database | MongoDB (Mongoose)            |
| Frontend | HTML, Tailwind CSS, jQuery    |

---

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- MongoDB running locally (with auth enabled)

### 1 – Install dependencies
```bash
cd backEnd
npm install
```

### 2 – Start the server
```bash
npm run server
```
Server starts on **http://localhost:3000**

### 3 – Open the frontend
Open `frontEnd/index.html` directly in your browser.

## 🔌 Socket.IO Events

### Client → Server (emit)

| Event         | Payload                                  | Description          |
|---------------|------------------------------------------|----------------------|
| `getPosts`    | *(none)*                                 | Fetch all posts      |
| `createPost`  | `{ title: string, content: string }`     | Create a new post    |
| `updatePost`  | `{ postId: string, title: string, content: string }` | Update a post |
| `deletePost`  | `postId: string`                         | Delete a post        |
| `searchPosts` | `searchTerm: string`                     | Search by title/content |

### Server → Client (listen)

| Event           | Payload                                          | Description                    |
|-----------------|--------------------------------------------------|--------------------------------|
| `posts`         | `Post[]`                                         | Response to `getPosts`         |
| `postCreated`   | `{ message: string, post: Post }`                | Broadcast on create            |
| `postUpdated`   | `{ message: string, postId, title, content }`    | Broadcast on update            |
| `postDeleted`   | `{ message: string, postId: string }`            | Broadcast on delete            |
| `searchResults` | `Post[]`                                         | Response to `searchPosts`      |
